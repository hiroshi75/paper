# Reproduction Guide

## Data
All pollen data from Neotoma Paleoecology Database (public). European data (331 sites) and ENA data (111 sites).

## Requirements
R >= 4.3 with neotoma2, vegan, parallel

## Scripts
- `paper8_indicator_sensitivity.R` — Five-indicator-set sensitivity analysis
- `paper8_misclassification.R` — Misclassification rate computation
