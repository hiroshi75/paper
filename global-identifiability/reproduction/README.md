# Reproduction Guide

## Data
All from Neotoma Paleoecology Database: Europe 331, ENA 111, China 45, LatAm 179, Africa 110 = 776 sites.

## Scripts
- `pilot_china_exceedance.R` — China/East Asia (71→45 sites)
- `pilot_latam_exceedance.R` — Central+South America (266→179 sites)
- `pilot_africa_exceedance.R` — Sub-Saharan Africa (176→110 sites)
- European and ENA analyses: see companion papers AAES-P-0003, AAES-P-0004
