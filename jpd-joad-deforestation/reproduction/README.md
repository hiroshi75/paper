# Reproduction Guide

## Overview
Reproduces analyses in "State Formation, Not Agriculture, Was Temporally Associated with Deforestation in Western Japan."

## Data
- Japanese Pollen Database (JPD) via Neotoma: 15 sites
- JOAD radiocarbon dates: 39,284 dates (publicly available)
- p3k14c global comparison: 179,689 dates

## Requirements
- R >= 4.3 with neotoma2, rcarbon
