# Reproduction Guide
## Data
154,466 radiocarbon dates from p3k14c database (Bird et al. 2022).
## Scripts
- `pilot_archaeological_network.R` — Initial 7-region SPD analysis and synchrony detection
- `paper11_full_analysis.R` — Distance-decay, bootstrap CIs, Bartlett correction, loess detrending, sub-regional analysis
- `paper11_review_response.R` — Taphonomic correction, climate proxy comparison, equal-sample subsampling, FDR correction
